# 07 — 进阶工作习惯（从"会用"到"用好"）

> **版本**：v1.0 (2026-06-01)
> **适用人群**：已走完新手学习路径，想提升效率的用户
> **阅读时机**：完成 Soul.md + 第一个连接器 + 第一个专家后

---

## 1. 工作空间分区法（输入前操作）

在输入指令**之前**，右下角选好/创建工作空间。不同项目放入不同空间，文档和对话独立存放，互不交叉污染。

```
做法：右下角工作空间 → 为每个项目创建独立空间 → 开始前先切换
```

**核心价值**：任务多时不混乱，每个项目的上下文独立保留。

---

## 2. 任务管理三件套（解放记忆）

- **重命名**：鼠标悬停任务 → 三点 → 重命名为有意义名称（不是"帮我写"）
- **保鲜膜**：每次操作自动保存为独立任务，锁定当时的上下文。下次打开，思路仍在原地
- **归档**：长期不用→右侧归档按钮收藏；头像→设置→数据管理→查看已归档任务

---

## 3. 探索模块一键复用（零门槛上手）

进入「探索」→ 搜索想做的事 → 点击「制作我的版本」。**无需写 Prompt，Skill 和专家已自动关联**，改关键词即可。

**核心心法**：不需要会写 Prompt，不需要知道用哪个 Skill 和专家——系统已全部关联好。

---

## 4. 专家 vs Skill 快速辨别

| | 专家 | Skill |
|--|------|-------|
| 本质 | 深度对话模式 | SOP+工具+脚本的能力扩展包 |
| 适合 | 需要聊背景、聊需求的场景 | 固定格式输出（PPT/PDF/图片/表格） |
| 举例 | "法律顾问"为你分析合同条款 | "PPT生成"直接产出演示文稿 |

---

## 5. Claw 远程工作流

通过 Claw 接入微信/QQ/企业微信 → 手机发指令 → 电脑在线执行（需保持开机）。

地铁上想到任务 → 微信发一句话 → 回到工位，结果已在桌面。

**核心价值**：工作随时在线，不被设备限制。

---

## 6. 产物分享（让协作零门槛）

**微信分享**（v4.22.14+）
产物生成后 → 点击「分享」→「微信」→ 生成二维码 → 同事扫码即看，无需安装 WorkBuddy。

**HTML 分享**（v4.22.15+）
产物 → 「导出为 HTML」→ 生成浏览器可打开链接 → 样式完整保留，适合正式交付。

**OAuth 应用管理**（v4.24.2+）
长期协作场景 → 头像 → 设置 → OAuth 应用管理 → 授权后团队可访问指定内容。

**适用场景**：周报分享、方案评审、客户交付、跨团队协作
