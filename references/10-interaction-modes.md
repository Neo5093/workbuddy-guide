# 10 — 交互模式与通道（Interaction Modes）

> **版本**：v1.0 (2026-06-04)
> **用途**：理解 WorkBuddy 的三种工作模式与三条交互通道，选对方式提效
> **来源**：WorkBuddy 官方教程 — Interaction 章节

---

## 三种工作模式（按权限递进）

把 WorkBuddy 当作新来的助手，三档放权范围：

```
Ask（问一问）→ Plan（想一想）→ Craft（做一做）
权限递增：只读 → 方案 → 直接动手
```

### Ask — 问一问（只看不动）

| 维度 | 说明 |
|------|------|
| **权限** | 仅查看文件并回答问题 |
| **可做** | 读取文件、分析数据、搜索信息、回答问题 |
| **不可做** | 修改文件、执行命令、写入任何内容 |
| **适合** | 调研阶段、阅读分析、确认理解、不熟悉的任务探索 |

### Plan — 想一想（先方案后执行）

| 维度 | 说明 |
|------|------|
| **权限** | 写分步计划，确认后再动手 |
| **可做** | 分析需求、设计方案、分解步骤 |
| **不可做** | 未经确认直接修改文件或执行命令 |
| **适合** | 复杂任务、高风险操作、需要对齐方案、多人协作 |

### Craft — 做一做（直接动手）

| 维度 | 说明 |
|------|------|
| **权限** | 拥有最大权限，直接操作文件、生成产物 |
| **可做** | 读写文件、执行命令、调用外部工具、生成成果物 |
| **注意** | 受权限模式约束（见下方），高风险操作仍需确认 |
| **适合** | 信任度高、任务明确、需要快速产出的场景 |

---

## 新手路径

```
Ask 熟悉功能 → Plan 看方案思路 → Craft 放手干
```

> ⚠️ **重要**：重要文件提前备份。切换到 Craft 前确认理解 Plan 的方案。

## 两种权限模式

控制 WorkBuddy 什么时候直接做、什么时候先问你。

### 默认权限（推荐）

| 操作类型 | 行为 |
|---------|------|
| 工作空间内读写 | 自动完成 |
| 写工作空间外文件 | 暂停确认 |
| 删除文件 | 暂停确认 |
| 执行脚本 | 暂停确认 |
| 调用系统级 API | 暂停确认 |

### 完全访问（谨慎使用）

- 减少弹窗确认，AI 自主决策增加
- ✅ 适合：可信、隔离、可恢复的环境，短时间使用
- ❌ 不建议：处理客户/财务/生产资料时开启

## 三条交互通道

交互不限于桌面端，三种通道按需选用，不互斥。

```
桌面端 Chat ──── 最完整：Ask/Plan/Craft 三档随手切
助理模式 IM ──── 人不在电脑前，手机一句话遥控本地
小程序 ──────── 云上常驻，电脑可关机
```

### 1. 桌面端（最完整）

在电脑前时的主力通道：
- 三档工作模式随意切换
- 可以使用所有文件、工具、连接器
- 完整上下文支持

### 2. 助理模式（IM 远程触发）

人不在电脑前时，从手机 IM 下指令，让本地电脑替你干活。

```
手机 IM → 服务端转发 → 本地电脑执行 → 产物回推 IM
```

| 关键点 | 说明 |
|--------|------|
| **本地必须开机** | 任务始终在本地电脑跑 |
| **工作目录固定** | 所有 IM 指令落在「助理」专属文件夹 |
| **单对话累积** | 同一通道所有远程指令在一个会话里 |

### 3. 微信小程序（口袋里的 AI）

和助理模式是扩展关系——多了一条云端常驻通道。

| 模式 | 特点 |
|------|------|
| **云上模式** | 任务在云端常驻执行，不依赖电脑开机 |
| **电脑模式** | 保留远程调用本地电脑的能力 |
| **定时任务** | 在小程序内创建/查看/暂停 Automations |
| **Skills + 专家** | 已安装 Skill / Experts / Expert Teams 能力对齐桌面端 |

### 助理 vs 小程序：二选一指南

| 对比 | 助理模式 | 小程序 |
|------|---------|--------|
| 执行位置 | 本地电脑 | 云端 / 本地 |
| 电脑要求 | 必须开机在线 | 云上模式可关机 |
| 核心价值 | 遥控本地资源 | 云上常驻 + 移动便携 |
| 选择标准 | **任务依赖本地资源** | **不需要本地资源 / 电脑关机时** |

---

## 三条通道怎么搭配

按任务依赖什么资源，选对应通道：

| 场景 | 用哪个 | 为什么 |
|------|--------|--------|
| 在电脑前 + 本地资源 | 桌面端 | Ask/Plan/Craft 三档随手切 |
| 出门临时来活儿 | 助理模式 | 手机 IM 一句话遥控本地电脑 |
| 长期定时 / 电脑关机 | 小程序 | 云上常驻，电脑无需开机 |

三档不互斥，按场景叠加用最顺手。
